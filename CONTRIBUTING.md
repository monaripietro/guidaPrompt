# Contributing

Grazie per il contributo!

## Regole rapide

- Mantieni lo stile: semplice, diretto, pratico; termini tecnici in inglese.
- Ogni nuova tecnica/affermazione deve avere almeno 1 esercizio che la testa.
- Gli esempi devono essere “vendor-neutral” (non legati a una singola chatbot).
- Preferisci template in `templates/` quando un pattern si ripete.

## Pull request checklist

- [ ] Ho aggiornato `docs/00-indice.md` (se necessario)
- [ ] Ho aggiunto/aggiornato esercizi in `exercises/`
- [ ] Ho incluso rubric + soluzione proposta + errori comuni
- [ ] Ho usato Markdown/XML per delimitare sezioni nei prompt d’esempio
